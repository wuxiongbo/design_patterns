# 1. 创建型

创建型设计模式

    主要解决 “对象的 创建” 问题

常用的有：
- 单例模式    design_patterns.chapter41 ~ design_patterns.chapter43
- **工厂模式** design_patterns.chapter44 ~ design_patterns.chapter45 
              （包括，简单工厂、工厂方法 和 抽象工厂）
- 建造者模式   design_patterns.chapter46

不常用的有：
- 原型模式    design_patterns.chapter47


# 2. 结构型

结构型设计模式

    主要解决  “类或对象的 组合 或 组装”  问题，
    此类模式，总结了一些  '类' 或 '对象' 组合在一起的经典结构

常用的有：
- **代理模式**   design_patterns.chapter48
- 桥接模式       design_patterns.chapter49
- **装饰者模式** design_patterns.chapter50
- **适配器模式** design_patterns.chapter51

不常用的有：
- 门面模式  无
- 组合模式  design_patterns.chapter53
- 享元模式  design_patterns.chapter54 ~ design_patterns.chapter55
