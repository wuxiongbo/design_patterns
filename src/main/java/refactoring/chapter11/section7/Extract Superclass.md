重复代码的某种形式就是：两个类以相同的⽅式做类似的事情，或者以不同的⽅式做类似的事情。

对象提供了⼀种简化这种情况的机制，那就是继承。

另⼀种选择, 就是 Extract Class （149）。
这两种⽅案之间的选择其实就是 继承 和 委托 之间的选择。
如果两个类可以共享⾏为，也可以共享接⼝，那么继承是⽐较简单的做法。
